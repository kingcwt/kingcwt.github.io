pug观后记

kingCwt · 2019-6-14 · 次阅读

看pug文档记录一些知识
  • 好记性不如烂笔头

  • 什么是pug，pug其实就是一个语法糖,好比写css 我们可以用less写一样 无非就是html换成啦pug,记录一下常用的pug语法坑以此勉励自己

  • 开始pug写作之前。一定要先 npm install pug 一下

  • 文档在这里 https://pug.bootcss.com/api/getting-started.html
了解一下

//- template.pug
p #{name}的 Pug 代码



const pug = require('pug');
const compiledFunction = pug.compileFile('template.pug');
// 渲染一组数据
console.log(compiledFunction({
  name: '李莉'
}));
// "<p>李莉的 Pug 代码！</p>"

// 渲染另外一组数据
console.log(compiledFunction({
  name: '张伟'
}));
// "<p>张伟的 Pug 代码！</p>"


----pug.renderFile 用法

const pug = require('pug');

// 编译并使用一组数据渲染 template.pug
console.log(pug.renderFile('template.pug', {
  name: 'Timothy'
}));
// "<p>Timothy 的 Pug 代码！</p>"



OK 了解一下  文档有！

在pug里用3元运算符

  • 提醒一下 注意代码的缩进 空格 
- var status=true
body(class=status?'authed':'toni')

==>html
<body class="authed"></body>

多行属性

input(
type='checkbox'
name='cwt'
checked
)
===> html

<input type='checkbox' name='cwt' checked>

变量存储

如果我们给一个标签写内容 就是空格 然后输入内容 比如
p 123
//这里的123就是内容最后编译成<p>123</p>
如果是变量呢？可以像下面这样
- var url='好好学习 天天向上';
p=url+'我是字符串'

//最后编译成=>
<p>好好学习 天天向上 我是字符串</p>
- 算是一个写法把。因为在后面和后台数据交互会用到

样式

p(style={color:"red",background:"blue"})

===>html
<p style='color:red;background:blue'></p>

简写标签和类名

1 如果是div 我们可以简写成这样 如果是类名我们用.来表示 ,如果是id我们用#来表示
  不是是类名还是id我们都依次写到后面就行 像下面这样 然后在加点属性
.b#c(style={color:"red",background:"blue"})
==>html
<div class='b' id='c' style='color:red;background:blue'></div>

case语句


- var friends=10
case friends
  when 0
    p 你没有朋友
  when 1
    p 你有1个朋友
  when 2
    break  
  default
    p='你有'+friends+'朋友'

===>html 
<p>你有10朋友</p>

- 如果你不想在2的时候输出东西 你可以break跳过这个语句

for循环

- for (var x = 0; x < 3; x++)
  li item

===>html

<li>item</li><li>item</li><li>item</li>

注释


//- - for (var x = 0; x < 3; x++)
  li item
//

unless 条件判断


- var a=4;
unless a>3
  p a大于3
else
  p none

===>html
<p>none</p>  


- var b=4;
if !a>3
  p a大于4
else
  p none

===>html
<p>none</p> 

- 所以 以上两种方式是相等的 其实就是if true取反

include

list.pug

header
  title 我的网址


doctype html
html
  include ./list.pug
  body
    h1 开始你的表演
    p come

===>html
<!DOCTYPE html><html><header><title>我的网址</title></header><body><h1>开始你的表演</h1><p>come</p></body></html>

- include 引入

extends 模版继承

//- layout.pug
html
  head
    title 我的站点 - #{title}
  body
      block content  


extends layout.pug
block content
  h1 123

===>html
<html><head><title>我的站点</title></head><body><h1>123</h1><div id="footer"><p>一些页脚的内容</p></div></body></html>

标签嵌入

p.
  这是一个很长很长而且还很无聊的段落，还没有结束，是的，非常非常地长。
  突然出现了一个 #[strong 充满力量感的单词]，这确实让人难以 #[em 忽视]。
p.
  使用带属性的嵌入标签的例子：
  #[q(lang="es") ¡Hola Mundo!]

===>html 
<p>这是一个很长很长而且还很无聊的段落，还没有结束，是的，非常非常地长。
突然出现了一个 <strong>充满力量感的单词</strong>，这确实让人难以 <em>忽视</em>。</p><p>使用带属性的嵌入标签的例子：
<q lang="es">¡Hola Mundo!</q></p>

each,while 循环（迭代）

  • Pug 目前支持两种主要的迭代方式： each 和 while。

each

ul
  each item in [1,2,3,4,5,6]
    li=item

===>html
<ul><li>1</li><li>2</li><li>3</li><li>4</li><li>5</li><li>6</li></ul>

- 可以加入索引
ul
  each val, index in ['〇', '一', '二']
    li= index + ': ' + val

===html 
<ul>
  <li>0: 〇</li>
  <li>1: 一</li>
  <li>2: 二</li>
</ul>    

 - 还可以迭代对象
 ul
   each val, index in {1:'一',2:'二',3:'三'}
     li= index + ': ' + val

===>html
<ul>
  <li>1: 一</li>
  <li>2: 二</li>
  <li>3: 三</li>
</ul>

- 还可以加个判断
- var values = [];
ul
  each val in values.length ? values : ['没有内容']
    li= val
===html
<ul>
  <li>没有内容</li>
</ul>

- 也可以用else来判断
- var values = [];
ul
  each val in values
    li= val
  else
    li 没有内容
===>html
<ul>
  <li>没有内容</li>
</ul>
  • 您也可以使用 for 作为 each 的别称。

while

- var n = 0;
ul
  while n < 4
    li= n++
===>html
<ul>
  <li>0</li>
  <li>1</li>
  <li>2</li>
  <li>3</li>
</ul>

Mixin(html复用)

  • 好似一个函数 然后可以传参数。在需要的地方引用
- 这种是不传参数的
mixin list
  ul
    li cwt
    li chr
    li ln
    li smd
+list
+list

===>html
<ul><li>cwt</li><li>chr</li><li>ln</li><li>smd</li></ul><ul><li>cwt</li><li>chr</li><li>ln</li><li>smd</li></ul>    

- 这种是传参数的
mixin pet(name)
  li.pet=name

ul
  +pet('猫')
  +pet('狗')
  +pet('猪')

===>html 
<ul><li class="pet">猫</li><li class="pet">狗</li><li class="pet">猪</li></ul>

* 这里的.pet是类名 name不是变量吗 =name就是把变量赋值给li 切莫混淆
  • Mixin 也可以把一整个代码像内容一样传递过来

mixin pet(name)
  .article
    .article-wrapper
      h1=name
      if block
        block
      else
        p 没有内容

+pet('我是h1')
  p 我是block内容

===>html

<div class="article"><div class="article-wrapper"><h1>我是h1</h1><p>我是block内容</p></div></div>

script标签

script.
  if (usingPug)
      console.log('这是明智的选择。')
    else
      console.log('请用 Pug。')

===>html

<script>
  if (usingPug)
    console.log('这是明智的选择。')
  else
    console.log('请用 Pug。')
</script>
